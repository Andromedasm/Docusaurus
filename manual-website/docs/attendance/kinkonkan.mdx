# 勤今管インストール手順

## ソフトウェアインストール

### 静紋ドライバーインストール
1. `JMNSDK_03_09`フォルダを開く
2. `Driver`フォルダを開く
3. `J300`フォルダを開く
4. `x64`フォルダを開く
5. `driver_setup_x64.exe`ファイルを実行
6. 全て`次へ`を選択し、ドライバーをインストール

### 静紋SDKインストール
1. `JMNSDK_03_09`フォルダを開く
2. `Runtime`フォルダを開く
3. `Installer`フォルダを開く
4. `x64`フォルダを開く
5. `JohmonSDK_x64.exe`ファイルを実行
6. `完全`を選択
7. `所属`を入力
8. `次へ`を選択し、SDKをインストール

### 静紋インストール
1. `JOHMON_01_11`フォルダを開く
2. `x64`フォルダを開く
3. `win8_10`フォルダを開く
4. `setup_win8_10_x64.exe`ファイルを実行
5. `カスタム`を選択
6. 以下の写真通りに選択 <img src="/img/kinkonkan/step4.png" />
7. `次へ`を選択し、静紋をインストール

### KinKonKanコピー
1. `KinKonKan`フォルダを選択し、コピーする
2. Cドライブの直下に貼り付け
3. `KinKonKan.exe`ファイルを右クリックして、デスクトップにショートカットを作成
4. `Windows+R`を押して、以下の内容を入力し、`OK`を選択
```shell
shell:startup
```
5. `KinKonKan.exe`ショートカットをコピーし、スタートアップフォルダに貼り付ける

## タスクスケジューラーの設定

1. `Windows+R`を押して、以下の内容を入力し、`OK`を選択
```shell
taskschd.msc
```
2. 左の`タスクスケジューラーライブラリ`→`Microsoft`→`Windows`を展開
3. `Time Synchronization`を選択
4. `SynchronizeTime`を右クリックし、`プロパティ`を選択<img src="/img/kinkonkan/synchroniz.png" />
5. `トリガー`を選択し、`新規`をクリック
6. 時刻同期スケジュールを以下のように設定<img src="/img/kinkonkan/new_toriga.png" />
```text
タスク開始設定:スケジュールに従う
毎日を選択
このスケジュールを実行する開始日を入力
毎日の何時に自動同期するかの時刻設定
間隔を1日に設定
「有効」にチェックを入れる
```
7. `OK`をクリック
8. `タスクの作成`を選択
9. 名前に`kinkonkan`を入力し、`最上位の特権で実行する`をチェック
10. `トリガー`を選択し、`新規`をクリック
11. 以下のように設定<img src="/img/kinkonkan/new_toriga2.png" />
```text
タスク開始設定:ログイン時
特定のユーザーを選択
「有効」にチェックを入れる
```
12. `操作`を選択し、`新規`をクリック
13. 以下のように設定<img src="/img/kinkonkan/new_sousa.png" />
14. `条件`を選択し、`コンピューターをAC電源で使用している場合のみタスクを開始する`のチェックを外す<img src="/img/kinkonkan/jyouken.png" />
15. `設定`を選択し、`タスクが失敗した場合の再起動間隔`を設定<img src="/img/kinkonkan/settei.png" />

## 勤今管動作確認
1. 勤今管を起動し、動作確認
2. PCを再起動し、勤今管起動できるか確認
