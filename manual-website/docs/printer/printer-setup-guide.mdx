# Windows 10/11 での Brother プリンター設定マニュアル

このガイドでは、Windows 10およびWindows 11でBrotherプリンターを設定する手順を詳細に説明します。プリンターを正しく設定することで、スムーズに印刷を行うことができます。

## プリンターの接続準備

1. **プリンターを開封**し、電源ケーブルを接続します。
2. **プリンターの電源を入れ**、Wi-Fi接続を使用する場合は、Wi-Fiネットワークに接続します（プリンターのディスプレイやマニュアルに従って設定してください）。

## ドライバーのインストール

### Brother公式サイトからドライバーをダウンロード

1. ブラウザを開き、Brotherの公式サイト（[Brotherサポートページ](https://support.brother.com)）にアクセスします。
2. 自分のプリンターのモデル番号を入力し、該当するモデルのサポートページに移動します。
3. 「ダウンロード」セクションに進み、使用しているOS（Windows 10またはWindows 11）を選択します。
4. 「フルドライバー＆ソフトウェアパッケージ」を選択し、ダウンロードを開始します。

### ドライバーのインストール手順

1. ダウンロードしたファイルをダブルクリックして実行します。
2. 画面の指示に従ってインストールを進めます。
3. インストールが完了したら、必要に応じてコンピューターを再起動します。

## プリンターの追加

### USB接続

1. USBケーブルを使用してプリンターをコンピューターに接続します。
2. Windowsが自動的にプリンターを検出し、インストールしたドライバーを使用してプリンターをセットアップします。

### Wi-Fi接続

1. **スタートメニュー**から「設定」を開きます。
2. 「デバイス」セクションに進み、「プリンターとスキャナー」を選択します。
3. 「プリンターまたはスキャナーを追加する」をクリックします。
4. システムがネットワーク上のプリンターを検索し、リストに表示されます。
5. 自分のBrotherプリンターを選択し、「デバイスを追加」をクリックします。

## プリンターの設定確認

1. **スタートメニュー**から「設定」を開きます。
2. 「デバイス」セクションに進み、「プリンターとスキャナー」を選択します。
3. インストールしたBrotherプリンターがリストに表示されていることを確認します。
4. プリンターを選択し、「管理」をクリックします。
5. 「プリンターのプロパティ」を選択し、プリンターの設定やステータスを確認します。

## トラブルシューティング

### プリンターが検出されない場合

1. **接続ケーブル**を確認し、正しく接続されていることを確認します。
2. **プリンターの電源**が入っていることを確認します。
3. **ネットワーク接続**を確認し、プリンターが同じネットワークに接続されていることを確認します。
4. 必要に応じて、**プリンターの再起動**を試みます。

### 印刷ができない場合

1. **プリンターのステータス**を確認し、エラーメッセージが表示されていないか確認します。
2. **インクまたはトナーの残量**を確認し、必要に応じて交換します。
3. **プリンタードライバー**が最新のバージョンに更新されていることを確認します。
4. **印刷キュー**を確認し、保留中のジョブがないか確認します。

### ドライバーのインストールがうまくいかない場合

1. **管理者権限**でインストーラーを実行します。
2. **セキュリティソフトウェア**を一時的に無効にし、再度インストールを試みます。
3. **Windows Update**を実行し、最新の更新プログラムを適用します。

以上で、Windows 10/11でのBrotherプリンターの設定手順についての説明を終わります。正しく設定することで、スムーズに印刷を行うことができます。何か問題が発生した場合は、上記のトラブルシューティングを参考にしてください。
