# Desknet's NEO 環境設定マニュアル

このガイドは、Desknet's NEOの環境設定に必要な情報を入力する手順を説明します。以下の設定を確認し、正しく入力してください。

## 環境情報の入力

### 本体インストール先
- `/var/www/cgi-bin/dneo`

### モバイルブラウザ版インストール先
- `/var/www/cgi-bin/dneosp`

### 安否確認回答機能インストール先
- `/var/www/cgi-bin/dneo/dneosafety`

### Webサーバーのホームディレクトリ
- `/var/www/html`

### データベース管理ユーザー名
- `postgres`

### データベース管理ユーザーパスワード
- `postgres`

### PostgreSQLインストール先
- `/usr/pgsql-13.3`

### 移行先のPostgreSQLポート番号
- `5433`

以上の設定を正しく入力することで、Desknet's NEOの環境が適切に構成されます。
