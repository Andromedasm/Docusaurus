# Markdown マニュアル

このガイドでは、Markdownの基本的な使い方から応用的なテクニックまでを詳細に説明します。Markdownは、軽量でシンプルな記法であり、HTMLに変換できるプレーンテキストフォーマットです。ブログ、ドキュメント、READMEファイルなどで広く使用されています。

## 基本的な文法

### 見出し
見出しを作成するには、`#` 記号を使用します。`#` の数で見出しのレベルを示します。

```markdown
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```

### 段落
段落は単純にテキストを行間に空行を入れずに続けて書くことで作成できます。

```markdown
これは段落です。

これは別の段落です。
```

### リスト
リストには番号付きリスト（順序付きリスト）と番号なしリスト（順序なしリスト）があります。

#### 順序なしリスト
```markdown
- アイテム1
- アイテム2
  - サブアイテム1
  - サブアイテム2
```

#### 順序付きリスト
```markdown
1. 最初のアイテム
2. 次のアイテム
 1. サブアイテム1
 2. サブアイテム2
```

### 引用
引用を作成するには、`>` 記号を使用します。

```markdown
> これは引用です。
> 複数行の引用を作成することもできます。
```

### 強調
テキストを強調するには、アスタリスク `*` またはアンダースコア `_` を使用します。

#### 斜体
```markdown
*斜体*
_斜体_
```

#### 太字
```markdown
**太字**
__太字__
```

#### 斜体と太字
```markdown
***斜体と太字***
___斜体と太字___
```

### コード
コードを表示するには、バックティック `` ` `` を使用します。

#### 行内コード
```markdown
これは `コード` です。
```

#### 複数行のコードブロック
```markdown
これは
複数行の
コードブロックです。
```

## リンクと画像

### リンク
リンクを作成するには、角括弧 `[]` と丸括弧 `()` を使用します。

```markdown
[リンクテキスト](https://example.com)
```

### 画像
画像を挿入するには、感嘆符 `!` と角括弧 `[]`、丸括弧 `()` を使用します。

```markdown
![代替テキスト](https://example.com/image.jpg)
```

## 表
表を作成するには、縦線 `|` とダッシュ `-` を使用します。

```markdown
| 見出し1 | 見出し2 |
| ------- | ------- |
| 内容1   | 内容2   |
| 内容3   | 内容4   |
```

## その他の機能

### 水平線
水平線を作成するには、3つ以上のダッシュ `-`、アスタリスク `*`、アンダースコア `_` を使用します。

```markdown
---
***
___
```

### 自動リンク
URLやメールアドレスをそのまま書くと、自動的にリンクになります。

```markdown
https://example.com
example@example.com
```

### エスケープ文字
Markdownの特殊文字をそのまま表示するには、バックスラッシュ `\` を使用します。

```markdown
\*これは斜体ではありません\*
```

これで、Markdownの基本的な使い方と応用的なテクニックについての説明を終わります。Markdownは非常に柔軟で強力なツールですので、ぜひ活用してください。
