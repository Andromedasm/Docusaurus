# Windows で Teams キャッシュをクリアする

このガイドは、WindowsでMicrosoft Teamsのキャッシュをクリアする手順を説明します。以下の手順に従ってください。

## Teams キャッシュのクリア手順

1. **Teams の終了**
- Teams がまだ実行中の場合は、サインアウトしたうえで、タスクバーの [Teams] アイコンを右クリックし、[終了] を選択します。

2. **「ファイル名を指定して実行」ダイアログボックスを開く**
- [Windows + R] キーを押して、「ファイル名を指定して実行」ダイアログボックスを開きます。

3. **%appdata% にアクセス**
- 「%appdata%」と入力し、[OK] をクリックします。

4. **「Microsoft」フォルダーを開く**
- 「Microsoft」フォルダーを開きます。

5. **「Teams」フォルダーを開く**
- 「Teams」フォルダーを開きます。

6. **「Cache」フォルダーを開く**
- 「Cache」フォルダーを開きます。

7. **キャッシュファイルの削除**
- すべてのファイルを選択し、削除します。

8. **Microsoft Teams の再起動**
- Microsoft Teamsを再起動します。

これで、Microsoft Teamsのキャッシュがクリアされ、正常に動作するはずです。
