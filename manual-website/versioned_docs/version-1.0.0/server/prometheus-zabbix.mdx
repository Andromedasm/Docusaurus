# Prometheus と Zabbix データの Grafana での統合マニュアル

このマニュアルでは、PrometheusとZabbixのデータをGrafanaで統合表示する方法について説明します。

## 前提条件

- Prometheus、Zabbix、およびGrafanaがインストールされ、正しく設定されていること。
- GrafanaにはPrometheusとZabbixのデータソースプラグインがインストールされていること。

## 1. Prometheus データソースの追加

1. Grafanaにログインし、「設定」＞「データソース」を選択します。
2. 「データソースの追加」をクリックし、「Prometheus」を選択します。
3. PrometheusサーバのURLを入力し、「保存 & テスト」をクリックします。

## 2. Zabbix データソースの追加

1. Grafanaの「データソース」ページに戻り、「データソースの追加」をクリックし、「Zabbix」を選択します。
2. Zabbixプラグインの設定画面で、ZabbixサーバのURL、ユーザー名、パスワードを入力します。
3. 「保存 & テスト」をクリックし、データソースの接続が成功したことを確認します。

## 3. ダッシュボードの作成

1. 「+」アイコン＞「ダッシュボード」を選択し、新しいダッシュボードを作成します。
2. 「パネルを追加」をクリックし、データソースとして「Prometheus」または「Zabbix」を選択します。
3. クエリを設定し、PrometheusまたはZabbixからのデータをグラフ化します。
4. 必要に応じて複数のパネルを追加し、PrometheusとZabbixのデータを1つのダッシュボードで表示します。

## 4. ダッシュボードのカスタマイズ

- ダッシュボードの見た目をカスタマイズするには、パネルのタイトル、グラフの種類、軸の設定などを変更します。
- アラートルールを設定して、特定の条件を満たしたときに通知を受けることができます。

これで、PrometheusとZabbixのデータをGrafanaで統合して表示する基本的な設定は完了です。必要に応じて、さらに詳細なカスタマイズや最適化を行ってください。
